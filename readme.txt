== Description ==

Check content and create a widget menu with links that point to words between the HTML tags you choose. 
It was initially thought and created for the HTML tags of the titles (h1, h2, h3â€¦), but you can use it in other situations.

== Installation ==

1. Upload the anchors-menu folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the Wordpress 'Widgets' menu move the widget to your sidebar
4. Select the options you want in the widget options panel

You are done! :-)
